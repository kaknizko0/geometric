# <font color="darkblue"> Geometric Lib </font>
## Общее описание решения
Библиотека позволяет работать с двумя фигурами: квадаратом и окружностью. С помощью нее можно посчитать периметр и площадь каждой фигуры.


# <font color="blue"> Math formulas </font>
## <font color="lightblue"> Area </font>
- Circle: S = πR²
- Square: S = a²
- Triangle: S = 1/2 * a * h
- Rectangle: S = a * b

## <font color="lightblue"> Perimeter </font>
- Circle: P = 2πR
- Square: P = 4a
- Triangle: P = a + b + c
- Rectangle: P = 2 * (a + b)

# <font color="blue"> Functions </font>

## <font color="lightblue"> Circle </font>
### 1. perimeter()
- Вычисляет длину окружности с заданным радиусом.

        Параметры:
            r (float): радиус окружности

        Возвращаемое значение:
            perimeter (float): длина окружности радиуса r

        Примеры:
            >>> area(5)
            <<< 78.53981633974483
            >>> area(2.5)
            <<< 19.634954084936208

### 2. area()
- Вычисляет площадь круга с заданным радиусом.

        Параметры:
            r (float): радиус круга

        Возвращаемое значение:
            area (float): площадь круга радиуса r

        Примеры:
            >>> perimeter(5)
            <<< 31.41592653589793
            >>> perimeter(2.5)
            <<< 15.707963267948966
       

## <font color="lightblue"> Square </font>
### 1. perimeter()
- Вычисляет периметр квадрата с заданной стороной.

        Параметры:
            a (float): длина стороны квадрата

        Возвращаемое значение:
            perimeter (float): периметр квадрата со стороной a 

       Примеры: 
            >>> area(5)
            <<< 25
            >>> area(3.5)
            <<< 12.25

### 2. area()
- Вычисляет площадь квадрата с заданной стороной.

        Параметры:
            a (float): длина стороны квадрата

        Возвращаемое значение:
            area (float): площадь квадрата со стороной a

        Примеры: 
            >>> perimeter(5)
            <<< 20
            >>> perimeter(3.5)
            <<< 14.0

## <font color="lightblue"> Triangle </font>
### 1. perimeter()
- Вычисляет периметр треугольника 

        Параметры:
            a (float) : длина первой стороны трегольника
            b (float) : длина второй стороны треугольника
            c (float) : длина третьей стороны треугольника

        Возвращаемое значение:
            float : периметр треугольника

        Примеры:
            >>> perimeter(5, 6, 4)
            <<< 15.0
            >>> perimeter(7, 2.5, 7.5)
            >>> 17.0

### 2. area()
- Вычисляет площадь треугольника

        Параметры:
            a (float) : длина стороны
            h (float) : высота треуголтника 

        Возвращаемое значение:
            float : площадь прямоугольника 

        Примеры:
            >>> arae(5, 6)
            <<< 15.0
            >>> area(5, 3.5)
            <<< 8.75

## <font color="lightblue"> Rectangle </font>
### 1. perimeter()
- Вычисляет периметр прямоугольника

        Параметры:
            a (float) : ширина прямоугольника
            b (float) : высота прямоуголника

        Возвращаемое значение:
            float : периметр прямоугольника

        Примеры:
            >>> perimeter(5, 10)
            <<< 30.0
            >>> perimeter(2.5, 7)
            <<< 19.0

### 2. area()
- Вычисляет площадь прямоугольника

        Параметры:
            a (float) : ширина прямоугольника
            b (float) : высота прямоуголника

        Возвращаемое значение:
            float : площадь прямоугольника

        Примеры:
            >>> area(5, 10)
            <<< 50
            >>> area(2.5 * 5)
            <<< 12.5
